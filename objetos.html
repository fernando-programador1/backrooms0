Objetos

…Objetos, também conhecidos como itens ou artefatos, são ferramentas, armas, consumíveis e outras coisas usadas ou encontradas pelas pessoas nos Backrooms.

Muitos deles seguem as regras da realidade normal, meras decorações criadas pelos níveis. Outros são objetos do mundo real alterados fundamentalmente pela estranheza dos Backrooms, e alguns são criados por exploradores para uso próprio ou para o uso de outros..

Alguns deles são perigosos, enquanto outros são cruciais para a sua sobrevivência. A identificação de objetos é extremamente importante para a habitação prolongada nos Backrooms, já que existem semelhantes.

A principal diferença entre uma entidade e um objeto é que as entidades estão vivas. Objetos indicados com [NO DATA] são endereços vazios no banco de dados, apenas esperando para serem preenchidos!

ALGUNS OBJETOS

Object 1 - "Almond Water"

Object 1 is a group of colored thermoses of liquid that can be found on various levels

Object 2 - "Level Keys"

Level Keys are mysterious objects that possess the ability to access certain levels through doors.

Object 3 - "Smiler Repellent"

The Lost have resulted in the discovery of a Rune directly pertaining to Smiler Repellent

Object 4 - "Deuclidator"

Deuclidators, also known as Warplooms, are small mechanical objects.

Object 5 - "Candy"

Object 5 is the collective designation of the various types of candy

Object 6 - "The Mirror"

Object 6 is a large, seemingly ordinary wall mirror hung underneath a green exit sign.

Object 7 - "Memory Jars"

Memory Jars are unique jars which can hold memories of the deceased.

Object 8 - "Lamps"

Lamps are a high tech device developed by Backrooms Robotics

Object 9 - "Dumb Gum"

Object 9, also commonly referred to as Dumb Gum, is a soft, that's visually similar to gum

Object 10 - "Scarecrows"

When traversing the wheat fields of Level 10, travelers will often come across large scarecrows.

Object 11 - "Ghost Lights"

Object 11 appears as a device referred to as a "Ghost Light".
